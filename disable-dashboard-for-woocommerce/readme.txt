== Frequently Asked Questions ==

= Do I need WooCommerce to use this plugin? =

No. The plugin does not require installing any additional plugins. There are many WooCommerce-oriented options, but you can just ignore them if you don't use WooCommerce.

= Will all my plugins be compatible? =

Some WooCommerce plugins rely heavily on WooCommerce Admin. If you decide to disable this module using our plugin, you may have a problem running these plugins:
- Google Listings & Ads

= I have a feature request =

Great! If you need a specific feature, please add a new thread in the plugin support forum. Then, your idea will be analyzed by the plugin team and may be introduced as a new feature in one of the upcoming releases.

== Installation ==

1. Upload the entire `disable-dashboard-for-woocommerce` folder to the `/wp-content/plugins/` directory (or install it using Add New Plugin)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the configuration page: Settings -> Disable Bloat
