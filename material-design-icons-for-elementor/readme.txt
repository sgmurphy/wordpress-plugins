== Description ==
Material Design Icons for Page Builders - adds Google Material Design Icons into Icons control of Page Builders

= Supported Page Builders: =
* Elementor
* Beaver Builder

= Shortcode Generator =
Generate shortcode to output an icon anywhere in the content.
