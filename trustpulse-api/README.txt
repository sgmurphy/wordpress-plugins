== Installation ==

1. Install TrustPulse either via the WordPress.org plugin repository or by uploading the files to your server.
2. Activate TrustPulse.
3. Click the button to set up a TrustPulse account, or log in to your existing account.
4. Click on the View my Campaigns button to start creating your FOMO social proof notifications.
