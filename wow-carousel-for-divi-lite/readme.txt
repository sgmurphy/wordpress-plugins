== Installation ==

> Note: This plugin works with Divi. Make sure you have [Divi Theme](https://www.elegantthemes.com/) installed.

Install the Torque Carousel for Divi either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.
