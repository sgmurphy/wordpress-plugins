== Description ==

51LA网站统计 WordPress 版本插件，快速引入到您的网站或博客中，用于统计网站访客、来路、事件分析和搜索引擎蜘蛛分析等，插件直接引入，支持 v5 / v6 版本网站统计，无需修改主题文件。

* **关于51LA** - 51.la 是免费的流量统计技术服务提供商，为互联网各类站点提供第三方数据统计分析。让站长了解网站现状，把握网站脉搏。

==  简单接入 ==

只需安装插件，注册并创建站点应用，按照配置页面提示填入ID到插件配置中，即完成代码安装接入，开启统计。

==  安装 ==

在线安装：

直接在 WordPress 管理后台 - 【插件】 - 【安装插件】，搜索 “51la Analysis”，点击安装即可。

手动安装：

点击本页右上角 Download 按钮下载，并将文件解压，把 51la-analysis 文件夹移动到 【wp-content】目录下的【plugins】目录内，并在 WordPress 后台中【插件】，找到 51la Analysis 启动。
