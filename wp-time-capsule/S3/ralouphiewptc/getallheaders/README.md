getallheaders
=============

This is a simple polyfill for [`getallheaders()`](http://www.php.net/manual/en/function.getallheaders.php).

## Install

For PHP version **`>= 5.6`**:

```
composer require ralouphiewptc/getallheaders
```

For PHP version **`< 5.6`**:

```
composer require ralouphiewptc/getallheaders "^2"
```
